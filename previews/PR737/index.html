Ferrite.jl

Welcome to the documentation for Ferrite.jl! Ferrite.jl is a finite element toolbox that provides functionalities to implement finite element analysis in Julia. The aim is to be i) general, ii) performant, and iii) to keep mathematical abstractions.

Note

Please help improve this documentation – if something confuses you, chances are you're not alone. How the documentation is organized

This high level view of the documentation structure will help you find what you are looking for. The document is organized as follows[1]:

  • Tutorials are thoroughly documented examples which guides you through the process of solving partial differential equations using Ferrite.jl.
  • Topic guides contains more in-depth explanations and discussions about the various concepts and ideas used in Ferrite.jl.
  • Reference contains technical reference of functions and APIs (e.g. the docstrings).
  • How-to guides will guide you through the steps involved in addressing common tasks and use-cases. These usually build on top of the tutorials and thus assume basic knowledge of how Ferrite.jl works.

Getting started

As a new user of Ferrite.jl it is suggested to start working with the tutorials before using Ferrite.jl to tackle the specific equation you ultimately want to solve. The tutorials start with explaining the basic concepts and then increase in complexity. Understanding the first tutorial program, solving the heat equation, is essential in order to understand how Ferrite.jl works. Already this rather simple program discusses many of the important concepts. See the tutorials overview for suggestion on how to progress to more advanced usage.

Getting help

If you have questions about Ferrite.jl it is suggested to use the #ferrite-fem channel on the Julia Slack, or the #Ferrite.jl stream on Zulip ([links to Slack and Zulip][community]). Alternatively you can use the discussion forum on the GitHub repository.

Installation

To use Ferrite.jl you first need to install Julia, see https://julialang.org/ for details. Installing Ferrite.jl can then be done from the Pkg REPL (press ] in the julia> promp to enter pkg> mode):

pkg> add Ferrite

This will install Ferrite.jl and Ferrite.jl's dependencies. Press backspace to get back to the julia> prompt. See the documentation for Pkg, Julia's package manager, for more help regarding package installation and project management.

Finally, to load Ferrite.jl, use

using Ferrite

You are now all set to start using Ferrite.jl!

Contributing to Ferrite.jl

Ferrite.jl is still under active development. If you find a bug, or have ideas for improvements, feel free to interact with the developers on the Ferrite.jl GitHub repository. There is also a thorough contributor guide which can be found in CONTRIBUTING.md.
